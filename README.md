# Second Brain - Flutter App

This is a Flutter port of the Second Brain Next.js application.

## Prerequisites

You need to have the Flutter SDK installed on your machine.
- [Install Flutter](https://docs.flutter.dev/get-started/install)

## Getting Started

1.  **Install Dependencies**
    ```bash
    flutter pub get
    ```

2.  **Run the App**
    ```bash
    flutter run
    ```

## Project Structure

- `lib/main.dart`: Main entry point.
- `lib/screens/`: Contains all UI screens (Landing, Login, Dashboard).
- `lib/widgets/`: Reusable UI components.
- `lib/theme/`: App styling using `AppTheme`.

## Troubleshooting

If you see `Command 'flutter' not found`, please ensure Flutter is installed and added to your system PATH.
